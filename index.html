Project on Console

Visual Studio Code is an IDE used for websites, web apps, and etc. To get Visual Studio click on this link: https://code.visualstudio.com/Download, and click on the computer you have. To start of make a new folder on your computer and open it on Visual Studio. Then create files for html, css, and js.

To open the console screen right click, and click "inspect". Then click on "console". The console screen is used to show an output. To print something on the console screen write "console.log("text");". If you want to do the same thing but print a variable, then do the same thing but inside the parentheses don't put an apostrophe.
